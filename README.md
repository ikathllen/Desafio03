## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Specifications

- Fazer as intalações.
- Criar um banco de dados: create database EMPRESA;
- Em '.env' deve colocar os dados de acesso do seu MySQL.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
